//Struktur Tag
<nama_tag atribut_tag="nilai_atribut_tag">
contoh : <head bgcolor="lightblue">
--- head        = tag
--- bgcolor     = warna background
--- "lightblue" = warna

boleh lebih dari 1 atribut, misal:
<head bgcolor="lighblue" id="myid" class="myclass">

//atribut global
- accesskey = menentukan shortcut keyboard
- class = menentukan class dari sebuah element
- id = menentukan id dari sebuah element
- dir = menentukan arah tulisan (direction)
- lang = menentukan bahasa (language)
- style = menyisipkan in-line CSS
- tabindex = menentukan urutan fokus cursor saat menekan "tab"
- title = memberi judul sebuah element
